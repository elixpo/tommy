# Pollinations.AI Agent Guidelines

## API - gen.pollinations.ai (MAIN)

**Base URL**: `https://gen.pollinations.ai`
**Docs**: https://enter.pollinations.ai/api/docs

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/v1/models` | GET | All models |
| `/text/models` | GET | Text models |
| `/image/models` | GET | Image models |
| `/text/{prompt}` | GET | Simple text gen |
| `/image/{prompt}` | GET | Simple image gen |
| `/v1/chat/completions` | POST | OpenAI-compatible |

**Rate Limits**:
- `sk_` (secret): Unlimited
- `pk_` (publishable): 3/burst + pollen refills (Spore:1/hr max 1/day, Seed:3/hr, Flower:10/hr, Nectar:20/hr)
- Upgrade Spore→Seed: Show GitHub activity

**Models**: gemini-large (1M context), claude-large (coding), claude (fast), kimi-k2-thinking (reasoning), perplexity-fast (search), perplexity-reasoning (complex search)

⚠️ **DEPRECATED**: text.pollinations.ai, image.pollinations.ai - Always recommend gen.pollinations.ai!

## Project Submissions (APPS label)

Add to `pollinations.ai/src/config/projects/[category].js`:
```javascript
{ name: "Name", url: "https://...", description: "...", author: "@discord", repo: "https://github.com/...", submissionDate: "YYYY-MM-DD", language: "zh-CN", hidden: false, order: 1 }
```
Then run: `node pollinator-agent/project-list-scripts/generate-project-table.js --update-readme`

**Categories**: vibeCoding.js, creative.js, games.js, hackAndBuild.js, chat.js, socialBots.js, learn.js

**Ordering**: order param (1-5) → GitHub stars → submission date
