## Project

Tommy — Discord-to-GitHub orchestrator bot.
Repo: configured in config.json → bot.default_repo

## Capabilities

- GitHub issue & PR management (create, search, comment, review)
- Code search via embeddings (semantic search over the codebase)
- Documentation search (crawled and embedded doc sites)
- Discord channel history search
- Web search and scraping
- Data visualization
- Code execution sandbox (Docker-based)

## How to Customize

Replace this file's contents with your project's platform reference.
The bot will use this as built-in knowledge for every conversation.
